**Para Method Overrides:**

- [ ] Método en clase base es `virtual` o `abstract`
- [ ] Firma del método **exactamente igual** (nombre, parámetros, retorno)
- [ ] Modificador `override` presente
- [ ] Llamas a `base.Metodo()` si necesitas lógica original
- [ ] No cambias el contrato del método (excepciones esperadas)
- [ ] Tipos de parámetros y retorno coinciden exactamente
- [ ] No agregas nuevas excepciones no documentadas
